# Public GitHub Launch Checklist

- Export only public packages and public docs.
- Verify public package boundary checks.
- Run clean install proof outside the monorepo.
- Confirm examples use placeholder ids and secrets.
- Confirm remote MCP endpoint docs match the deployed backend.
- Confirm package READMEs do not reference private app modules.
- Publish SDK, pairing CLI, MCP server, runtime hook packages, Guardian, and
  runtime adapters with provenance.
