# Contributing

Thanks for your interest in Filepad Agent Access.

This repository contains the public SDK, pairing CLI, and runtime adapters only. The Filepad hosted app, backend, frontend, billing, deployment, and internal automation systems are not part of this repository.

Before opening a pull request:

```bash
pnpm install
pnpm validate:agent-access:release
```

Please do not include real workspace ids, integration key ids, Agent Access secrets, logs, screenshots with secrets, or private Filepad deployment details.
